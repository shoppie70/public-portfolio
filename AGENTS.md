# Antigravity エージェント指示書 (sho-tsukamoto.jp)

本ファイルは、このリポジトリで作業する AI エージェント（Antigravity等）に対する指示書および開発ルールです。エージェントは作業を開始する前に必ずこのファイルを読み込み、指示に従ってください。

---

## 1. プロジェクト概要
本プロジェクトは、塚本 翔（Sho Tsukamoto）氏のポートフォリオサイト（https://sho-tsukamoto.jp）のフロントエンドソースコードです。

### 技術スタック
- **ビルドツール/バンドラー**: Vite (v8.1.2)
- **CSSフレームワーク**: Tailwind CSS (v3.4.19)
- **CSSプリプロセッサ**: Sass (SCSS)
- **JSフレームワーク/ライブラリ**:
  - Alpine.js (v3.15.12): 軽量なリアクティブ処理用
  - Swiper (v14.0.1): カルーセル/スライダー
  - Animate.css (v4.1.1): アニメーション用
  - Axios (v1.18.1): APIリクエスト用
  - Lodash (v4.18.1): ユーティリティ

---

## 2. ディレクトリ構造
主要なディレクトリとファイルの構成は以下の通りです。

- `index.html`: ポートフォリオのメインHTMLファイル。
- `assets/`:
  - `css/app.scss`: メインのSassスタイルシート。Tailwindディレクティブ等もここに記述。
  - `js/app.js`: エントリポイントとなるJavaScript。
  - `js/bootstrap.js`: ライブラリ等の初期化処理。
- `tailwind.config.js`: Tailwind CSSの設定。
- `postcss.config.js`: PostCSSの設定（Autoprefixer, TailwindCSS）。
- `dist/`: `npm run build` で生成される静的ファイル出力先。

---

## 3. 開発およびビルド手順
コードを変更した際は、必ず以下のコマンドを実行し、ビルドエラーが発生しないことを確認してください。

- **ローカル開発サーバー起動**:
  ```bash
  npm run dev
  ```
- **ビルド確認**:
  ```bash
  npm run build
  ```
